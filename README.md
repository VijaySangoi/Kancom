# Kancom
*this project is tribute to many many people's who have been great help in my journey.*

Kancom is an e-commerce solution built with intention to provide one platform for e-sellers from where they can manage there multiple e-commerce stores. its designed to facilitate integration of several e-commerce stores via API's provided by respective platform. current version supports ebay API's with future plan to expand support to other ecommerce platform. platform also supports multiple suppliers integration ie:- factory outlets and warehouses whom software can put orders in auto mode\*

following are key features of this software
 *  make it easier for sellers to manage multiple stores on different e-commerce sites
 *  let sellers place orders to suppliers once an order & payment is received on e-commerce store
 *  provide secure way to exchange data between e-commerce store, seller and supplier
 *  provide a way to update quantity and price daily
